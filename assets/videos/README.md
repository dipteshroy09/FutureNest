# Project Demo Video

Upload your demo video here or add a link to the external video in README.md.

## Guidelines

- **File naming**: `futurenest-demo.mp4` or `demo-video.mp4`
- **Recommended length**: 2-3 minutes
- **Max file size for GitHub**: 100MB

## Alternative: External Hosting

If your video exceeds 100MB, upload to:
- **YouTube** (unlisted) — most reliable
- **Google Drive** (shareable link)
- **Loom** — quick sharing
- **Vimeo**

Then add the link to the main README.md in the Video Demo section.
